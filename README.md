# Cisco Packet Tracer Labs

This repository contains hands-on Cisco Packet Tracer labs I completed as part of my networking and cybersecurity training.

## Skills Demonstrated
- Network design and topology creation
- Connecting network devices (routers, switches, PCs)
- IP addressing and subnetting
- DHCP IP address release and renew
- Layer 7 traffic generation
- Traffic analysis using Packet Tracer Simulation Mode

## Tools Used
- Cisco Packet Tracer
- TCP/IP
- DHCP
- OSI Model
- Simulation Mode traffic analysis

## Labs Included
1. Basic Network Setup
2. IP Addressing & DHCP Configuration
3. Traffic Generation & Analysis

Each lab folder contains:
- `.pkt` file
- Network topology screenshots
- Step-by-step documentation
